Java Script

Html

È uma linguagem de marcação de hipertexto, ele permite inserir o conteúdo e estabelecer a estrutura básica de um website.

Css

È a sigla para Cascading Style Sheets, ou seja, Folhas de Estilo em Cascatas. É uma maneira de dar estilo ao código criado por linguagens como HTML, por exemplo. De forma prática, ela funciona como uma camada de personalização ao conteúdo visível.

React JS

È uma biblioteca JavaScript criado em 2011 pelo time do Facebook, o React surgiu com o objetivo de otimizar a atualização e a sincronização de atividades simultâneas no feed de notícias da rede social, entre eles chat, status, listagem de contatos e outros.

Git

È um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software, mas pode ser usado para registrar o histórico de edições de qualquer tipo de arquivo

Git-Hub

È o maior repositório de dados compartilhado do mundo, com 28 milhões de usuários atualmente.uma plataforma majoritariamente usada por desenvolvedores, pois permite uma hospedagem prática de código-fonte e arquivos em nuvem.
